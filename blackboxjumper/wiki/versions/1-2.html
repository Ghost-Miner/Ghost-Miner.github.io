1.2 is a major Windows edition update released on 10th June 2020.


Additions

  • Added screenshake
  • Added double jump
  • Added hoover efect to menu buttons
  • Added advanced options  (Experimental)
    • You can modify size screen shake effect
    • You can modify player movement
  • Added Play options menu

 

Changes

  • Play Tutorial and Level select moved to extra menu Play options
  • Changed tutorial music
  • Game starts in fullscreen by default

 

Fixes and other

  • Fixed sticking to walls bug (may be re-added backin the future as a wall jump)
  • Better Game over screen
  • Other game optimizations
