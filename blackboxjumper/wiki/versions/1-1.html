1.1 is a major Windows edition update released on 6th June 2020.


Additions

  • Added play options to the game. Pressing "Play" button now give you 4 options - Continue, New game, Select level and play tutorial.
  • Added "Continue" feature. Press Continue in play options to start where you quit last time.
  • Added 2 new levels.
  • Added hoover effect for all buttons in menu.
  • Added particle effect when gate opens.
  • Added particle effect when enemy is killed.

Changes

  • Level buttons now turn grey if the level isn't unlocked.
  • "Continue" and "Level select" buttons turn grey when the game is run for the first time or if game progress was reset.
  • Changed and/or improved some technical stuff which isn't visible in game
  • "Play tutorial" and "level select moved to play options"
