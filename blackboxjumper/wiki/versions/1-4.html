1.4 is a major Windows edition update released on 29th January, 2021, focused on revamping the saving system.


Additions

Game saving and file system

  • All game settings are now saved in a file, so the settings don’t get reset every time the game is run.
  • Level information (attempts, current level etc.) is also now saved in a file.
  • All files are now saved in corresponding folders instead of in the game’s root folder.
  • Game progress is now saved in single file instead of 29 individual empty files.
  • Added visual feedback when you take a screenshot.
  • Added icons to main menu buttons.
  • Full keyboard control and navigation:
  • Navigate through the game and menus without using the mouse.
  • Press Alt to show keyboard shortcuts or go to settings to enable it .

Settings update:

  • Settings divided into categories.
  • Added more settings.
  • Toggle between game’s or Windows‘ mouse pointer.
  • Enable keyboard/no mouse navigation.

Changes

  • Report button moved below Quit button.
  • Code optimizations.
  • Cheats/Debug mode update:
  • Levels don’t save while cheats are enabled.
  • Level countdown turns yellow when there’s less than 5 seconds left.
  • Changed what’s shown on the tutorial completed screen.
  • Screenshake cannot go in negative numbers.

Fixes

  • Fixed F2 – screenshot and F11 – fullscreen not working in some cases.
    Fixed Level numbers – Level 1.01 => Level Easy 1.

Removals

  • Score counter.
  • Advanced options screen.
  • Leftover sprites, scripts and other unused junk.
