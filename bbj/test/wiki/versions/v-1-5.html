1.5 is a major Windows edition update released on 31st May 2021 which added new levels and fixed link buttons not working in Chromium browsers.


Additions

  • Confirmation dialog to reset progress button 
  • Reset settings button (also with confirmation dialog box)
  • Added Open speedrun record folder button
  • Added Open custom levels folder button
  • Added 8 new easy levels

Changes

  • Open folder buttons now open corresponding folder in system’s file explroer 

Fixes

  • Fixed links on working in chromium browsers
